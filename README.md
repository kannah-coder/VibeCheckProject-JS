# 🌟 Vibe Check Mini Project

A fun and interactive web app that matches weather and moods with unique “vibes.”  
Built using **HTML**, **CSS (Bootstrap + custom styling)**, and **JavaScript** with the OpenWeatherMap API.

## 📂 Project Structure
- `index.html` → Main page with Bootstrap layout  
- `app.js` → JavaScript logic (gallery, vibes, weather API calls)  
- `style.css` → Custom CSS for extra styling  
- `images/` → Contains vibe images (`chill.jpg`, `hype.jpg`, `cozy.jpg`, `confused.jpg`, `salty.jpg`, `boujee.jpg`)  

---

## ⚡ Features
- 🌤 Fetches real-time weather using OpenWeatherMap API  
- 🎭 Shows different “vibes” (Chill, Hype, Cozy, Boujee, etc.)  
- 💅 Responsive design with Bootstrap + custom styling  
- 🖼 Image-based vibe cards for fun interaction  

---

Add your images inside the images/ folder with names:
chill.jpg, hype.jpg, cozy.jpg, confused.jpg, salty.jpg, boujee.jpg

## Open app.js and replace:

 - const apiKey = "YOUR_API_KEY_HERE";


- with your OpenWeatherMap API key.

Run locally by simply opening index.html in your browser.
